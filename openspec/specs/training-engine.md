# Spec: Training Engine

CradleCoach 执行功能训练引擎的交互规范。

## 训练游戏

| 游戏 | 训练维度 | 规则 |
|------|----------|------|
| 反义词 | 反应抑制 | 选词 → 等待回答 → 规则判断；水果例外规则增加难度 |
| 倒序记忆 | 工作记忆 | 3-6 个词 → 等待倒序复述 → 规则判断；词数量 + 抽象度控制难度 |
| 情绪猜谜 | 情绪识别 | 内置故事库选段 → 等待识别情绪 → 规则判断 |
| 故事接龙 | 认知灵活性 | 生成开头 → 等待续写 → 评估长度和连贯性 |

## 约束

1. 训练以对话中嵌入的小游戏形式存在，非独立课程
2. 每次 8-15 分钟，每周 3-5 次
3. 难度自适应调节（基于正确率和响应时间）
4. 每次训练结束有明确结束仪式 + 社交引导："去找爸爸妈妈分享一下"
5. 正面反馈聚焦努力过程而非结果（"我看到你很专注"而非"你真聪明"）
